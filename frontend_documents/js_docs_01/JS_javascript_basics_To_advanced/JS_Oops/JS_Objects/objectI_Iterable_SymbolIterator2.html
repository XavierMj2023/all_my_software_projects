Symbol Iterator - It can use for of

When you start a for...of loop over myNumbers, JavaScript internally checks if myNumbers is iterable by checking if it has a Symbol.iterator property.

If myNumbers has a Symbol.iterator property, JavaScript accesses it to retrieve an

iterator object.

The iterator object returned by myNumbers[Symbol.iterator] is then used by the

for...of loop to iterate over the elements/values of myNumbers.

Each iteration of the loop calls the iterator's next() method to retrieve the next value, and the loop continues until the iterator indicates that the iteration is complete (i.e., when the done property of the iterator's result is true).

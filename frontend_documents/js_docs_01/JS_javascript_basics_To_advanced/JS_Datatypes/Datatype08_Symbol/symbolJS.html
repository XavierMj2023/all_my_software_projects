JavaScript Datatypes Demonstartion

  • JavaScript is a dynamic type language, means you don't need to specify type of the variable because it is dynamically used by JavaScript engine.

  • There are two types of data types in JavaScript.

    1. Primitive data type
    2. Non-primitive (reference) data type

  • JavaScript has dynamic types. This means that the same variable can be used to hold different data types.

8 Symbol
  • Represents a unique and immutable value that may be used as the key of an Object property.

  • Introduced in ES6 (ECMAScript 2015), symbols are immutable and are used as unique identifiers for object properties.

  • Each time you create a symbol, it is unique.

  • Symbols are a special data type introduced in ECMAScript 6 (ES6) that represent unique identifiers.

  • They cannot be implicitly converted to strings, as they are distinct from string values. However, you can explicitly convert a symbol to a string using the toString() method.

  let sym = Symbol('unique');

JavaScript Datatypes Demonstartion

  • JavaScript is a dynamic type language, means you don't need to specify type of the variable because it is dynamically used by JavaScript engine.

  • There are two types of data types in JavaScript.

    1. Primitive data type
    2. Non-primitive (reference) data type

  • JavaScript has dynamic types. This means that the same variable can be used to hold different data types.

7 Null
  • Represents a null value or a variable with no value.

  • It represents the absence of a value or a lack of a meaningful result.

  • Represents the intentional absence of any object value.

  • It is used to indicate that a variable points to no object.

  • In JavaScript, null is considered a primitive, despite being represented as the null object.

*   The purpose of a web browser (Chrome, Edge, Firefox, Safari) is to read HTML documents and display 
    them correctly.

*   A browser does not display the HTML tags, but uses them to determine how to display the document.

*   The content inside the <body> section will be displayed in a browser. 
    The content inside the <title> element will be shown in the browser's title bar or in the page's 
    tab.

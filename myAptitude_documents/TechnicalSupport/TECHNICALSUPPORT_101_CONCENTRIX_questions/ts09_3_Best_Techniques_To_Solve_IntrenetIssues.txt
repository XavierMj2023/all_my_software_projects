----------------------------------------------------------------------------------------------------------------------------------------------------
Solving an internet issue can involve several steps, but here are three of the most effective and commonly recommended steps to troubleshoot and resolve basic internet connectivity problems:
----------------------------------------------------------------------------------------------------------------------------------------------------

----------------------------------------------------------------------------------------------------------------------------------------------------
Step 1: Check Physical Connections and Restart Devices
----------------------------------------------------------------------------------------------------------------------------------------------------
Verify Physical Connections:
----------------------------------------------------------------------------------------------------------------------------------------------------
Cables		: Ensure that all cables (Ethernet, coaxial, power) are securely connected to your modem, router, and computer.

Lights		: Check the indicator lights on your modem and router. Typically, there should be a solid power light, a solid or blinking internet light, and activity lights for connected devices.

----------------------------------------------------------------------------------------------------------------------------------------------------
Restart Devices:
----------------------------------------------------------------------------------------------------------------------------------------------------
Modem and Router	: Unplug the power cables from both your modem and router. Wait for about 30 seconds, then plug the modem back in first. Wait until the modem is fully powered on (all relevant lights are stable), then plug in the router.

Computer or Device	: Restart your computer or the device experiencing connectivity issues.

----------------------------------------------------------------------------------------------------------------------------------------------------
Step 2: Diagnose Network and Software Settings
----------------------------------------------------------------------------------------------------------------------------------------------------
Network Settings:

Wi-Fi			: Ensure your device is connected to the correct Wi-Fi network and that the network is not set to "public" if it should be private.
IP Address		: Check if your device is receiving a proper IP address. 
			  On Windows, you can use the Command Prompt with the command -> ipconfig /all. 			  
			  On macOS, use the Terminal with the command ->  ifconfig.

----------------------------------------------------------------------------------------------------------------------------------------------------
Software Settings:
----------------------------------------------------------------------------------------------------------------------------------------------------

Network Adapter		: Disable and then enable your network adapter. On Windows, go to Network Connections, right-click your adapter, and select "Disable," then "Enable."

Firewall and Antivirus	: Ensure that your firewall or antivirus software is not blocking your internet connection. Temporarily disable them to see if the issue is resolved.
----------------------------------------------------------------------------------------------------------------------------------------------------

----------------------------------------------------------------------------------------------------------------------------------------------------
Step 3: Test and Reset Network Configurations
----------------------------------------------------------------------------------------------------------------------------------------------------
Ping and Connectivity Tests:
----------------------------------------------------------------------------------------------------------------------------------------------------
Ping Test		: Use the command prompt or terminal to ping a reliable website (e.g., ping google.com). If you get responses, the connection to the 			  internet is likely working, but there might be other issues.

Traceroute		: Run a traceroute to identify where the connection might be failing (e.g., tracert google.com on Windows, traceroute google.com on macOS).
----------------------------------------------------------------------------------------------------------------------------------------------------
Reset Network Configurations:
----------------------------------------------------------------------------------------------------------------------------------------------------
DNS Settings		: Change your DNS settings to use a public DNS server like Googleâ€™s (8.8.8.8 and 8.8.4.4).

Network Reset		: On Windows, you can perform a network reset to reinstall network adapters and set networking components back to their default settings. Go to Settings > Network & Internet > Status > Network reset.

Router Reset		: As a last resort, reset your router to factory settings. Note that this will erase all custom settings, so you may need to reconfigure your network afterward.

----------------------------------------------------------------------------------------------------------------------------------------------------
Additional Tips:
----------------------------------------------------------------------------------------------------------------------------------------------------
Check for Service Outages	: Sometimes, the issue is with your Internet Service Provider (ISP). Check their website or contact their support to see if there are any known outages.

Update Firmware and Drivers	: Ensure that your modem/router firmware and network adapter drivers are up-to-date.

Try a Different Device		: If possible, connect a different device to your network to see if the issue is specific to one device.

----------------------------------------------------------------------------------------------------------------------------------------------------
By following these three steps, you can address most common internet connectivity issues efficiently. If the problem persists, contacting your ISP for further assistance may be necessary.
